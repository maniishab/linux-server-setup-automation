# Linux Server Setup and Automation

This is a beginner DevOps project where I set up a Linux server and created simple automation scripts using Bash.

The goal of this project was to understand how basic server administration works and how repetitive tasks can be automated.

## What I Did

- Installed Ubuntu Server
- Learned basic Linux commands
- Managed packages using apt
- Created Bash scripts for automation
- Implemented a backup system
- Monitored system resources
- Scheduled tasks using cron

## Scripts

update.sh  
Automates system updates using apt.

backup.sh  
Creates a compressed backup of the home directory.

monitor.sh  
Displays CPU, memory, and disk usage.

## Technologies Used

Linux (Ubuntu Server)  
Bash Scripting  
Cron Jobs

## What I Learned

- Linux file system basics
- File permissions
- Package management
- Writing simple Bash scripts
- Automating tasks using cron
